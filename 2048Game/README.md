## 源码目录介绍
```
./js
├── base                                   // 定义游戏开发基础类
│   └── sprite.js                          // 游戏基本元素精灵类
├── libs
│   ├── storage.js                         // localStorage封装
│   └── weapp-adapter.js                   // 小游戏适配器
├── runtime
│   ├── background.js                      // 背景类
│   ├── gamehelp.js                        // 用于展示帮助界面
│   ├── gamelayer.js                       // 用于展示游戏主界面
│   ├── gameover.js                        // 用于展示分数和结算界面
├── databus.js                             // 管控游戏状态
└── main.js                                // 游戏入口主函数

```
